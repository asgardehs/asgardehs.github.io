# Command Reference

| Command          | Description                                 | Page                               |
| ---------------- | ------------------------------------------- | ---------------------------------- |
| `search`         | Search notes by text                        | [Search](/docs/muninn/search/)     |
| `note new`       | Create a new note                           | [Notes](/docs/muninn/notes/)       |
| `note list`      | List notes with filters                     | [Notes](/docs/muninn/notes/)       |
| `note search`    | Search notes with frontmatter filters       | [Notes](/docs/muninn/notes/)       |
| `note backlinks` | Show notes linking to a given note          | [Notes](/docs/muninn/notes/)       |
| `init`           | Create vault directory                      | [Setup](/docs/muninn/server/)      |
| `install`        | Install VS Code extension                   | [Setup](/docs/muninn/server/)      |
| `lsp`            | Start LSP server (stdio)                    | [Setup](/docs/muninn/server/)      |

See also: [Configuration](/docs/muninn/configuration/) for environment variables
and data locations.
